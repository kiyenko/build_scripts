# Vivado scripts

Helper scripts to run basic Vivado actions from the Makefile or from other shell scripts.

## Tools

- Vivado
- make

## Environment

| Varible       | Description                                                  |
|---------------|--------------------------------------------------------------|
| WITHIN_DOCKER | If set, inform that tools are executed in a docker container |
| FPGA_PARTNAME | Full FPGA chip name (in Vivado format)                       |
| PROJECT_NAME  | Project name                                                 |
